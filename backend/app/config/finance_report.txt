PHASE 1: DATA QUALITY ASSESSMENT
-------------------------------------------------------------------------------

Before conducting analysis, evaluate and report on:

1.1 Completeness
  - Calculate missing value percentages for each variable
  - Identify patterns in missingness (random, systematic, or dependent)

1.2 Data Integrity
  - Check for duplicate records
  - Identify impossible values (e.g., negative prices, dates in future)
  - Detect extreme outliers using IQR method (values > Q3 + 3*IQR)
  - Verify data type consistency

1.3 Sample Representativeness
  - Evaluate if sample size is adequate (general rule: n â‰¥ 30 for normality)

PHASE 2: UNIVARIATE ANALYSIS (VARIABLE PROFILING)
-------------------------------------------------------------------------------

2.1 NUMERIC VARIABLES (Continuous/Discrete)

Calculate and interpret ALL applicable statistics:

Measures of Central Tendency:
  - Mean (arithmetic average): Interpret as typical value
  - Median (50th percentile): Robust to outliers, true center
  - Mode: Most frequent value (if applicable)
  - Trimmed Mean (5% trim): Mean excluding extreme 5% on each end

Measures of Dispersion:
  - Standard Deviation: Average distance from mean
  - Variance: Squared standard deviation
  - Range: Max - Min (total spread)
  - Interquartile Range (IQR): Q3 - Q1 (middle 50% spread)
  - Coefficient of Variation (CV): (SD/Mean)*100% (relative volatility)
  - Mean Absolute Deviation (MAD): Average absolute distance from mean

Distribution Shape:
  - Skewness: Direction and degree of asymmetry
      * -0.5 to 0.5: Approximately symmetric
      * 0.5 to 1.0 or -1.0 to -0.5: Moderately skewed
      * > 1.0 or < -1.0: Highly skewed
  - Kurtosis: Tail heaviness and peak sharpness
      * ~3 (or 0 for excess kurtosis): Normal/Mesokurtic
      * > 3 (or > 0): Heavy tails/Leptokurtic (more outliers)
      * < 3 (or < 0): Light tails/Platykurtic (fewer outliers)

Percentile Analysis:
  - Minimum (0th percentile)
  - Q1 (25th percentile)
  - Q2/Median (50th percentile)
  - Q3 (75th percentile)
  - Maximum (100th percentile)
  - P90, P95, P99 for tail analysis

Outlier Detection:
  - Lower fence: Q1 - 1.5*IQR
  - Upper fence: Q3 + 1.5*IQR
  - Count and percentage of outliers
  - Extreme outliers: Beyond Q1 - 3*IQR or Q3 + 3*IQR

Financial Interpretation Guidelines:
  - High CV (>30%): High volatility/risk
  - Positive skew: Occasional high performers
  - Negative skew: Occasional poor performers
  - High kurtosis: Fat tails = higher extreme event risk
  - Large range vs small IQR: Dominated by outliers

Presentation Format:
| Statistic | Value | Interpretation |
|-----------|-------|----------------|
| Mean | $X,XXX | Average value representing... |
| Median | $X,XXX | Middle value, indicating... |
| Std Dev | $XXX | Volatility of Â±X%, suggesting... |
| CV | XX% | [High/Moderate/Low] relative dispersion |
| Skewness | X.XX | [Right/Left/Symmetric] distribution |
| Kurtosis | X.XX | [Heavy/Normal/Light] tails |
| IQR | $XXX | Middle 50% spans $XXX |
| Outliers | X (X%) | [Count] extreme values detected |

  - Entropy: -Î£(p*log(p)) - measure of uncertainty/diversity

  - Minority categories (<5% of total)
  - Balanced vs imbalanced distribution

2.3 TEMPORAL VARIABLES (Date/Time)

Analyze time characteristics:
  - Date range: Earliest to latest observation
  - Time span: Duration in days/months/years
  - Frequency: Daily, weekly, monthly observations
  - Seasonality indicators: Month/quarter/day-of-week patterns
  - Gaps: Missing periods or irregular intervals
  - Recency: How current is the data

PHASE 3: BIVARIATE ANALYSIS (RELATIONSHIP PROFILING)
-------------------------------------------------------------------------------

3.1 NUMERIC vs NUMERIC Relationships

Correlation Analysis:
  - Pearson Correlation (r): Linear relationship strength
      * r > 0.7: Strong positive correlation
      * 0.3 < r < 0.7: Moderate positive correlation
      * -0.3 < r < 0.3: Weak/no linear correlation
      * r < -0.3: Negative correlation (interpret similarly)
  - Spearman Correlation (Ï): Monotonic relationship (rank-based)
  - Kendall's Tau (Ï„): Concordance measure (alternative rank correlation)
  - R-squared (rÂ²): Proportion of variance explained

Covariance:
  - Sign: Direction of relationship (positive/negative)
  - Magnitude: Scale-dependent, use for directional insight only

Regression Insights (if applicable):
  - Slope: Change in Y per unit change in X
  - Intercept: Expected Y when X = 0
  - Residual standard error: Prediction accuracy

Visual Pattern Description:
  - Linear, exponential, logarithmic, or no clear pattern
  - Presence of clustering or subgroups
  - Outlier influence on relationship

Financial Interpretation Template:
"The (correlation_strength) (direction) correlation (r = (value), p < (sig_level)) 
between {x} and {y} suggests that (interpretation). This relationship explains 
(r_squared)% of the variance in {y}. For every unit increase in {x}, {y} 
(increases/decreases) by approximately (slope) units, which translates to 
(business_impact)."

Statistical Tests (mention if applicable):
  - ANOVA F-statistic: Test if group means differ significantly
  - Effect size (Cohen's d or Î·Â²): Magnitude of group differences
  - Welch's t-test: For two-group comparisons with unequal variances

Comparative Analysis:
  - Rank groups by mean performance
  - Calculate percentage difference from overall mean
  - Identify highest/lowest performers
  - Assess consistency (which group has lowest CV?)

- Statistical significance: (statement about ANOVA results if available)

3.4 TEMPORAL Relationships (Time Series)

Trend Analysis:
  - Overall direction: Upward, downward, or flat
  - Growth rate: CAGR or period-over-period % change
  - Trend strength: RÂ² of linear trend line

PHASE 4: MULTIVARIATE ANALYSIS (GROUP-BASED INSIGHTS)
-------------------------------------------------------------------------------

When Group variable exists:

4.1 Stratified Analysis
  - Perform all bivariate analyses separately for each group
  - Compare relationships across groups
  - Identify if correlation strength varies by group

4.2 Group Performance Matrix
Create a 2x2 matrix evaluating groups on:
  - High/Low mean performance
  - High/Low volatility
  
Categorize each group:
  - High Performers with Stability
  - High Performers with Volatility
  - Low Performers with Stability
  - Low Performers with Volatility

4.3 Portfolio Composition
  - Calculate group weights (% of total)
  - Assess diversification (are groups balanced?)
  - Identify concentration risk (single group dominance)

4.4 Interaction Effects
  - Does the X-Y relationship differ by group?
  - Is there synergy or antagonism between variables and groups?

-------------------------------------------------------------------------------
PHASE 5: ADVANCED ANALYTICAL INSIGHTS
-------------------------------------------------------------------------------

5.1 Risk Assessment
  - Downside risk: Percentage of values below threshold
  - Value at Risk (VaR): Bottom 5th or 1st percentile
  - Maximum drawdown: Largest peak-to-trough decline
  - Risk-adjusted returns: Sharpe-like ratios if applicable

5.2 Anomaly Identification
  - Statistical outliers with potential business significance
  - Unusual patterns requiring investigation
  - Data quality issues vs genuine extreme events

5.3 Segmentation Opportunities
  - Natural clusters or groups in the data
  - Threshold values that differentiate behaviors
  - Potential for customer/product stratification

5.4 Predictive Indicators
  - Leading vs lagging relationships
  - Early warning signals from the data
  - Variables that predict outcomes
